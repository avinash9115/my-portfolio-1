Project 1

The project is a basic card game implemented using vanilla JavaScript, basic CSS, and HTML code. It covers various concepts such as animation effects, responsive layout design using CSS grid, CSS flexbox, and media queries, as well as dynamically changing the positions of grid cells to randomize the positions of the cards. The game involves four different cards, including the ace of spades. When the user clicks the "Play Game" button, the cards are flipped over, hiding their faces. The game then shuffles the positions of the cards, randomly placing them in a grid layout using JavaScript and local storage. The user can then guess the position of the ace of spades by clicking on a card. Once the user makes a choice, the selected card flips over, revealing its face, and the user is informed whether their guess was correct or not.
